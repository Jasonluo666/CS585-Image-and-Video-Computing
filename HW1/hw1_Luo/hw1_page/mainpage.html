Assignment 1 Description

The main purpose of the assignment 1 is to do some simple processing on the image of our face. The assignment page refers to here.

  1. Convert the image into grayscale
  2. Flip the image horizontally
  3. add a blur effect on the image

Showcase

Original Image

"This is the original image of my face which can be used to compare with the processed images."
Grayscale Image

"In order to creating a grayscale image of my face, I define a new image object which has only one color channel (original RGB image has three color channels).

Then, I used the formula: gray_channel_value = 0.3*R + 0.6*G + 0.1*B to generate the values for the grayscale image."
Flipped Image

"This image is horizontal flip of the original image. The method I took is to reverse all the values in each row of the image matrix. More specifically, I used the pointers to achieve this: swap(row[index], row[column - index])"
Blurred Image

"The blurred image is created by using a simple algorithm:
for every pixel, assign it with the average value of its neighbors (usually 8 neighbors).

After 10 times iterations, I got this blurred image."
